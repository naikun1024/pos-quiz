品詞を見分けるコツ（接尾辞でほぼ判別できる）

英単語の多くは語尾（接尾辞）で品詞が見分けられます。これを覚えるのが最速ルートです。

  • 名詞：-tion / -sion / -ment / -ness / -ity / -er / -or / -ist / -ance / -ence
    例：information, decision, development, happiness, ability, teacher, actor, artist
  • 動詞：-ize / -ise / -ify / -ate / -en
    例：realize, classify, communicate, strengthen
  • 形容詞：-ful / -less / -ous / -ive / -al / -ic / -able / -ible / -ant / -ent
    例：useful, careless, famous, active, natural, economic, reliable
  • 副詞：-ly（ただし friendly, lovely は形容詞なので注意）
    例：quickly, carefully, immediately

※レベル区分は市販教材で頻出する単語を目安として分類したもので、ETS公式の分類ではありません。

💡 プログラミング学習アドバイス（クリックで開く）

このアプリはHTML/CSS/JavaScriptの3つだけで作られています。ソース（右クリック→ページのソース表示）を読んでみると勉強になります。

  • データと表示の分離：単語リスト（WORDS）とUI描画を別々に書いています。これが基本。
  • 状態管理：stateオブジェクトに「今どのレベルか／何問目か／正解数は」を集約。Reactなどでも同じ考え方です。
  • 次のステップ：このアプリを改造してReact/TypeScriptで書き直すと、モダンフロントの練習に最適。
